## 블로그를 시작하며

드디어 블로그를 시작합니다. 오랫동안 생각만 해왔던 개인 블로그를 Next.js로 만들게 됐습니다.

## 사용 기술 스택

이 블로그는 다음 기술들로 만들어졌습니다:

- **Next.js 16** — App Router 기반의 프레임워크
- **TypeScript** — 타입 안전성
- **Tailwind CSS v4** — 스타일링
- **Velite** — MDX 콘텐츠 관리
- **Shiki** — 코드 문법 하이라이팅

## 코드 예시

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

앞으로 개발하면서 배운 것들을 꾸준히 기록할 예정입니다.
